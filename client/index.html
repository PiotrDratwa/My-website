Writing quality projections AI

The goal of this project was to create an AI that would predict students’ essay grades based on the data of the essay (how many individual cases were used, how many capital letters were there) and students’ behaviour during the essay. We had to transform the data and use different machine learning algorithms to find the best working one. Finally the AI had mean error of around 0.4 on the scale 2-5.5.
The technologies used: Python, Seaborne, Pandas, sci-kit learn.

dataframe encryption project

It’s a Python library made in C, to encrypt pandas dataframes using the RSA algorithm. It also includes example of usage. The reason it is made in C is because Python computes very slowly and all popular fast libraries for Python are actually made in C.
